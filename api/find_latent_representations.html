step 1: find_latent_representation#

usage: GPS run_find_latent_representations [-h] --input_hdf5_path
                                           INPUT_HDF5_PATH --output_hdf5_path
                                           OUTPUT_HDF5_PATH --sample_name
                                           SAMPLE_NAME
                                           [--annotation ANNOTATION]
                                           [--type TYPE] [--epochs EPOCHS]
                                           [--feat_hidden1 FEAT_HIDDEN1]
                                           [--feat_hidden2 FEAT_HIDDEN2]
                                           [--feat_cell FEAT_CELL]
                                           [--gcn_hidden1 GCN_HIDDEN1]
                                           [--gcn_hidden2 GCN_HIDDEN2]
                                           [--p_drop P_DROP] [--gcn_lr GCN_LR]
                                           [--gcn_decay GCN_DECAY]
                                           [--n_neighbors N_NEIGHBORS]
                                           [--label_w LABEL_W] [--rec_w REC_W]
                                           [--n_comps N_COMPS]
                                           [--weighted_adj] [--nheads NHEADS]
                                           [--var]
                                           [--convergence_threshold CONVERGENCE_THRESHOLD]
                                           [--hierarchically]

Named Arguments#

--input_hdf5_path

Path to the input hdf5 file.

--output_hdf5_path

Path to the output hdf5 file.

--sample_name

Name of the sample.

--annotation

Name of the annotation layer.

--type

The gene expression layer of the input data (e.g., ‘count’, ‘counts’, ‘log1p’).

--epochs

Number of training epochs for the GNN-VAE model. Default is 300.

Default: 300

--feat_hidden1

Number of neurons in the first hidden layer of the feature extraction network. Default is 256.

Default: 256

--feat_hidden2

Number of neurons in the second hidden layer of the feature extraction network. Default is 128.

Default: 128

--feat_cell

Number of top variable genes to select. Default is 3000.

Default: 3000

--gcn_hidden1

Number of units in the first hidden layer of the GCN. Default is 64.

Default: 64

--gcn_hidden2

Number of units in the second hidden layer of the GCN. Default is 30.

Default: 30

--p_drop

Dropout rate used in the GNN-VAE model. Default is 0.1.

Default: 0.1

--gcn_lr

Learning rate for the GCN network. Default is 0.001.

Default: 0.001

--gcn_decay

Weight decay (L2 penalty) for the GCN network. Default is 0.01.

Default: 0.01

--n_neighbors

Number of neighbors to consider for graph construction in GCN. Default is 11.

Default: 11

--label_w

Weight of the label loss in the loss function. Default is 1.

Default: 1

--rec_w

Weight of the reconstruction loss in the loss function. Default is 1.

Default: 1

--n_comps

Number of principal components to keep if PCA is performed. Default is 300.

Default: 300

--weighted_adj

Use a weighted adjacency matrix in GCN. Default is False.

Default: False

--nheads

Number of heads in the attention mechanism of the GNN. Default is 3.

Default: 3

--var

Enable var. Use –var to enable. Default is False.

Default: False

--convergence_threshold

Threshold for convergence during training. Training stops if the loss change is below this threshold. Default is 1e-4.

Default: 0.0001

--hierarchically

Find latent representations hierarchically. Use –hierarchically to enable. Default is False.

Default: False
